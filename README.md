# 英语阅读器

一个网页版英语阅读器，可以上传英语文档或粘贴文本进行阅读，支持全文/整段/选中内容朗读，并能在点击单词时展示字典小卡片、发音和翻译。

## 功能

- 上传 TXT、Markdown、HTML 文档并在本地解析。
- 上传 PDF、DOCX 文档时，浏览器会通过 CDN 加载 PDF.js 或 Mammoth.js 解析。
- 输入或粘贴英语文本后载入阅读区。
- 支持全文朗读、整段朗读、选中文本朗读、暂停/继续和停止。
- 可选择朗读语言/声音、调整语速；慢速档会自动按标点切句并加入停顿，让听感真正变慢。
- 阅读区支持全文翻译，译文会按段落显示在独立面板中，方便对照英文正文。
- 点击英语单词会自动发音并显示字典卡片，包含发音、音标、词性、英文释义、每个词性最多一句例句和经过筛选的目标语言词义；点击页面空白处会自动关闭卡片。

## 本地运行

直接打开 `index.html` 即可使用主要功能。

也可以使用 Node 启动本地服务器：

```bash
node server.mjs
```

然后访问 `http://localhost:4173`。

## 说明

朗读功能使用浏览器内置的 Web Speech API。字典查询使用 `dictionaryapi.dev`。词义优先使用内置高可信常见词表和词形还原结果，在线翻译结果会经过匹配度、语言脚本和长度过滤；低可信结果不会直接当作准确译义展示。
