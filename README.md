# 英语阅读器

一个网页版英语阅读器，可以上传英语文档或粘贴文本进行阅读，支持全文/整段/选中内容朗读，并能在点击单词时展示字典小卡片、发音和翻译。

## 功能

- 上传 TXT、Markdown、HTML 文档并在本地解析。
- 上传 PDF、DOCX 文档时，浏览器会通过 CDN 加载 PDF.js 或 Mammoth.js 解析。
- 输入或粘贴英语文本后载入阅读区。
- 支持全文朗读、整段朗读、选中文本朗读、暂停/继续和停止。
- 可选择朗读语言/声音、调整语速。
- 点击英语单词显示字典卡片，包含发音、音标、英文释义和目标语言翻译。

## 本地运行

直接打开 `index.html` 即可使用主要功能。

也可以使用 Node 启动本地服务器：

```bash
node server.mjs
```

然后访问 `http://localhost:4173`。

## 说明

朗读功能使用浏览器内置的 Web Speech API。字典查询使用 `dictionaryapi.dev`，翻译查询使用 MyMemory 翻译接口；在网络不可用时会回退到内置常见词表。
